TP N° 1

// Contexte — Notation des nombres romains La notation des nombres romains est basée sur l'utilisation des lettres M, D, C, L, X, V et I. L'équivalent décimal du nombre Romain se calcule selon le principe suivant :
◆ L'équivalent décimal de chaque chiffre romain est : M=1000, D=500, C=100, L=50, X=10, V=5 et I=1.
◆ L'équivalent décimal de la chaîne de chiffres romains est obtenu en additionnant les équivalents décimaux de ses chiffres. Le parcours de la chaîne se fait de gauche à droite et dans le cas où un chiffre est inférieur à son successeur, il sera précédé du signe moins (−).
Exemple :
RCDXL
CDXL −100+500−10+50
"CDXL" = −100 + 500 − 10 + 50 = 440

On se propose de développer une application (interface et code) qui permet de saisir une chaîne R non vide contenant uniquement des chiffres romains, Calculer et d'afficher l'équivalent décimal de R.

Travail demandé :
  1. Concevoir une interface comme illustré ci-contre et l'enregistrer dans votre dossier de travail sous le nom « interface_romain.ui ».
  2. Créer un programme Python et l'enregistrer dans votre dossier de travail sous le nom « romain.py ».
  3. Dans le programme « romain.py », implémenter en Python la fonction chiffre(c).
  4. Développer, dans le programme « romain.py », une fonction decimal(R) qui permet calculer et retourner l'équivalent décimal de la chaîne composée des chiffres romain R.
  5. Dans le programme « romain.py » :
    • ajouter les instructions permettant d'appeler l'interface graphique intitulée « interface_romain.ui »,
    • développer un module qui s'exécute à la suite d'un clic sur le bouton "Equivalent", permettant de récupérer le nombre romain R, exploiter la fonction 'decimal' afin de calculer et afficher l'équivalent décimal de la chaîne R via le label désigné à l'affichage dans l'interface « interface_romain.ui ».
Romain – Décimal
Introduire un nombre romain :
CDXL
Equivalent
CDXL équivaut en décimal 440
Fonction chiffre (c : caractère) : entier
Début Selon c faire 'M' : q ← 1000 'D' : q ← 500 'C' : q ← 100 'L'  : q ← 50 'X' : q ← 10 'V' : q ← 5 'I'  : q ← 1 Fin Selon Retourner q Fin
NB : Afficher le message d'erreur « Nombre romain invalide » si la donnée récupérée R est incorrecte.
